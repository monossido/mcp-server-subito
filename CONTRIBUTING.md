# Contributing

Contributions are welcome! This project is community-driven and every bit of help is appreciated.

## How to contribute

- **Bug reports and feature requests** — open an [issue](https://github.com/monossido/mcp-server-subito/issues)
- **Pull requests** — fork the repo, make your changes, and open a PR against `master`
- **Questions and ideas** — feel free to reach out at [monossido@lorenzobraghetto.com](mailto:monossido@lorenzobraghetto.com)

## Guidelines

- Keep changes focused — one feature or fix per PR
- Make sure the project builds (`npm run build`) and tests pass (`npm test`) before submitting
- Follow the existing code style

By contributing you agree that your work will be licensed under the same [AGPL v3](LICENSE) license as this project.
